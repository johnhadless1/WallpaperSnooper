## Features "what does it do?"

- It just changes your wallpaper automatically
- Able to remember what tags you want for next wallpapers (For example, "mountains","cyan","towerwatch",etc...)
- Change every day / week / custom (IN SECONDS!!) / unlock
- Preview wallpapers in app (this is show you what future wallpapers will look when you let it work)
- You can change advanced settings such as resolution, type, etc... whatever man
- It uses your system color palette and ___Material 3 You___!!!!!! (ok whatever, but at least it looks cool!!)
  
## Permissions "this is so corporate man"

- Able to boot on start up (you can turn this off via settings)
- Able to change your currect wallpaper/background
- Able to display notification (this is "just in case" that you forgot you had WallpaperSnooper running in background plus can be turned off via settings)
- Run in background or run as foreground (you can turn this off via settings)
